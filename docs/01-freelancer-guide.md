# Freelancer Guide

## Quick Start

### 1. Untuk Penawaran Baru
1. Copy folder quotations
2. Edit file yang sesuai (web/mobile/fullstack)
3. Ganti [placeholder] dengan data client
4. Export ke PDF atau send via WA/Email

### 2. Untuk Kontrak
1. Copy contracts/01-project-contract-template.md
2. Fill in semua detail
3. Both parties sign

---

## Best Practices

### Communication
- Response dalam 24 jam
- Tone professional tapi friendly
- Document everything

### Pricing
- Fixed price, bukan hourly
- Include contingency
- Don't underprice

### Delivery
- Clear milestones
- Regular updates weekly
- Deliver more than promised

### Support
- Don't abandon after delivery
- Offer maintenance package
- Ask for testimonials

---

## Common Mistakes

| Mistake | Impact |
|---------|--------|
| No written contract | Risk tidak dibayar |
| Unclear scope | Rework |
| No milestones | Cash flow problems |
| Undercharging | Burnout |
