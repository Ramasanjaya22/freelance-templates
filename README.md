# Freelance Templates

Template quotations, contracts, dan proposals untuk pekerjaan freelance.

## Struktur

```
freelance-templates/
├── quotations/      # Template quotation harga
├── contracts/      # Template kontrak & SLA
├── proposals/      # Template proposal teknis
├── price-list/     # Daftar harga standar
├── portfolio/      # Template showcase project
└── docs/          # Panduan penggunaan
```

## Cara Pakai

1. **Quotations** - Copy folder quotations, edit sesuai kebutuhan client
2. **Contracts** - Sesuaikan nama, harga, dan terms sebelum tanda tangan
3. **Proposals** - Custom untuk pitch ke client besar
4. **Price List** - Referensi pricing untuk estimasi cepat
